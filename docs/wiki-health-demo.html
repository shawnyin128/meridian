  • Deterministic first.
    Default health does not call an LLM.

    Scores roll up stable subchecks under Trust, Surface, Context, Graph, and Growth.
  • Hard fails override score.
    Source contamination or draft retrieval blocks release.

    Every finding maps to a mechanism-level repair path.
  • Stable benchmark cases.
    Retrieval usefulness uses fixed case sets.
  • Agent optional.
    Agent review consumes top-k repair candidates only.
